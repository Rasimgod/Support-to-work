Анализ работы сервера Linux:
top: Показывает текущие процессы и их использование ресурсов.
htop: Интерактивный аналог top с более удобным отображением информации о процессах.
free -h: Показывает использование оперативной памяти.
df -h: Показывает использование дискового пространства.
netstat -tuln: Показывает активные сетевые соединения и открытые порты.
iotop: Показывает активные процессы, которые выполняют операции ввода-вывода на диске.
dmesg: Показывает сообщения ядра о событиях и ошибках.
journalctl -xe: Показывает подробные сообщения системного журнала о произошедших событиях.

Администрирование приложений, демонов и служб:
systemctl start service_name: Запускает службу.
systemctl stop service_name: Останавливает службу.
systemctl restart service_name: Перезапускает службу.
systemctl status service_name: Показывает состояние службы.
systemctl enable service_name: Включает автозапуск службы при загрузке системы.
systemctl disable service_name: Отключает автозапуск службы при загрузке системы.
systemctl list-unit-files --type=service: Показывает список всех установленных служб и их состояние (включено/выключено).
journalctl -u service_name: Просмотр журналов для конкретной службы.
