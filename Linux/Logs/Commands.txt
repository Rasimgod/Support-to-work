Поиск в файлах:
grep pattern /path/to/file: Поиск строки или шаблона в файле.
grep -r pattern /path/to/directory: Рекурсивный поиск строки или шаблона в указанном каталоге и его подкаталогах.
find /path/to/directory -name "*.txt" -exec grep pattern {} \;: Поиск по файлам с заданным расширением в указанном каталоге и его подкаталогах с использованием команды grep.

Редакторы текстовых файлов:
Nano: Простой текстовый редактор с основными функциями.
nano /path/to/file: Открывает файл для редактирования в редакторе Nano.
Vim: Мощный текстовый редактор с широким набором функций.
vim /path/to/file: Открывает файл для редактирования в редакторе Vim.
В режиме вставки: нажмите i, чтобы начать вставку текста.
В режиме команд: нажмите Esc, чтобы выйти из режима вставки, затем :wq для сохранения и выхода, :q! для выхода без сохранения изменений.
Emacs: Текстовый редактор с расширенными возможностями.
emacs /path/to/file: Открывает файл для редактирования в редакторе Emacs.
Нажмите Ctrl + X Ctrl + S для сохранения изменений, Ctrl + X Ctrl + C для выхода.

Просмотр логов:
cat /var/log/messages: Показывает системные сообщения.
cat /var/log/secure: Показывает аутентификационные сообщения и журналы аудита.
cat /var/log/audit/audit.log: Показывает журналы аудита.
journalctl: Просмотр журналов системного журнала.
journalctl -u service_name: Просмотр журналов для конкретного службы.
journalctl -p level: Просмотр журналов для конкретного уровня журналирования (например, -p err для ошибок).
journalctl --since "2022-01-01" --until "2022-01-02": Показать журналы с определенной даты и времени.

Управление журналами:
journalctl --vacuum-size=500M: Удаляет старые записи из журналов системного журнала для уменьшения размера.
journalctl --rotate: Поворачивает текущие файлы журналов системного журнала.

Анализ логов:
grep keyword /var/log/file.log: Поиск ключевого слова в лог-файле.
tail -f /var/log/file.log: Отображение последних строк лог-файла в реальном времени.
less /var/log/file.log: Постраничное отображение лог-файла с возможностью прокрутки вверх и вниз.
awk '{print $1}' /var/log/file.log: Извлечение определенных полей из лог-файла с помощью AWK.

Архивирование и ротация логов:
logrotate: Утилита для архивирования и ротации лог-файлов.
logrotate -f /etc/logrotate.conf: Принудительно запускает ротацию лог-файлов.

Управление журналами аудита:
auditctl: Управление настройками аудита и добавление правил аудита.
ausearch: Поиск в журналах аудита с помощью различных критериев.
aureport: Генерация отчетов на основе журналов аудита.
